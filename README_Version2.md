# ArraysPlus

Napredni blokovi za rad s nizovima (number[]) u MakeCode micro:bit.

Kako koristiti:
1. Stvori novi GitHub repozitorij i dodaj datoteke `pxt.json`, `api.ts` i `README.md`.
2. Otvori https://makecode.microbit.org
3. Klikni Settings (zupčanik) → Extensions → u polje zalijepi:
   - github:YOUR_GITHUB_USERNAME/YOUR_REPO_NAME
   ili cijeli URL repozitorija.
4. Nakon uvoza u Toolboxu će se pojaviti kategorija "ArraysPlus" s blokovima.

Primjer korištenja u Blocks editoru:
- Kreiraj varijablu tipa Array (Number).
- Koristi blokove ArraysPlus: stavi (push), ukloni zadnji (pop), dohvati/set element, sortiraj, promiješaj, zbroj, prosjek, jedinstveni elementi, itd.

Napomena:
- Ova ekstenzija radi s nizovima brojeva (number[]). Ako želiš rad s tekstualnim nizovima (string[]), mogu dodati i te blokove.
- Ako želiš drugačije nazive blokova (npr. na engleskom), javi i prilagodit ću ih.
